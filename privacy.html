Política de Privacidad - ThinkTwice

Última actualización: 15 de agosto de 2025

1. Datos que recopilamos

  • URLs visitadas: ThinkTwice accede a la dirección web que estás a punto de visitar únicamente para mostrar un mensaje de reflexión antes de entrar al sitio.
  • Configuración de usuario: El usuario no tiene configuraciones para añadir por ahora

2. No recopilamos

  • Información personal identificable (nombre, correo, dirección, etc.)
  • Contenido de las páginas web visitadas (texto, imágenes, vídeos, enlaces)
  • Contraseñas, datos financieros, información sanitaria o comunicaciones privadas

3. Uso de los datos

Las URLs se usan únicamente localmente para decidir cuándo mostrar el mensaje de reflexión.

4. Compartir información

ThinkTwice no comparte ni vende tu información a terceros.
